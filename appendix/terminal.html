Terminal Basics

You can use the terminal to do all sorts of things. Here are the basics skills this resource covers:

  1. Make a folder
  2. Enter the folder
  3. Make a file in the folder
  4. Edit the file
  5. Delete the file
  6. Delete the folder

The commands for the terminal are slightly different depending on the operating system you use. Use the appropriate resource below to learn some basic terminal commands!

Windows Command Line Tutorial

Command Description Example
pwd Print the name of the working directory, your “location” the terminal is currently focused on pwd
dir Print out the contents of the current directory dir
cd [folder name] Change directory; “move” the terminal to focus on a different location. Must be one of the options displayed by dir or an absolute path. To move “up” to the parent folder, type cd .. cd images
mkdir [name] Make a folder in the working directory mkdir images
type nul > [filename] Create a file in the working directory type nul > README.md
rm [filename] Delete a file rm README.md
rmdir \S [folder name] Delete a folder and its contents rmdir \S images

Mac/Linux Terminal Tutorial

Command Description Example
pwd Print the name of the working directory, your “location” the terminal is currently focused on pwd
ls Print out the contents of the current directory ls
cd [folder name] Change directory; “move” the terminal to focus on a different location. Must be one of the options displayed by ls or an absolute path. To move “up” to the parent folder, type cd .. cd images
mkdir [name] Make a folder in the working directory mkdir images
touch [filename] Create a file in the working directory touch README.md
rm [filename] Delete a file rm README.md
rm -rf [folder name] Delete a folder and its contents rm -rf images
nano [file name] Edit a file in the terminal using the nano text editor. When using the editor, type Ctrl+S to save, and Ctrl+X to exit nano README.md
